latar belakang :Di lerai/kedai, pemesanan manual sering menyebabkan antrean, salah catat pesanan, dan kurang efisien. Karena itu dibuat sistem pemesanan makanan berbasis web yang bisa diakses lewat HP.

tujuan project : Mempermudah pelanggan memesan tanpa ke kasir
Mengurangi kesalahan pesanan
Mempercepat pelayanan
Menerapkan konsep web responsif

konsep/sistem pemesanan :
1. Pelanggan buka web yang sudah disediakan barcode di setiap meja atau di media sosial
2. Pilih menu makanan atau minuman
3. Pesanan diproses otomatis
4. Admin melihat pesanan dari sistem
5. pembayaran menyediakan qris di setiap meja
6. pemesanan di antar sesuai pesanan di meja nomer yang di isi

teknologi berbasis :
HTML struktur halaman
CSS tampilan & responsif di HP
JavaScript interaksi (hitung total, tombol)
PHP proses data & penyimpanan
Database (MySQL) menyimpan pesanan

fitur :
login
Daftar menu makanan
pilihan minuman dingin atau panas
Keranjang pesanan
Total harga otomatis
Form pemesanan
Halaman admin (lihat pesanan)
